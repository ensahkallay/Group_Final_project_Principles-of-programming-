# EduMaster School Management System

## Overview

EduMaster School Management System is a desktop application developed using Python Tkinter. The system helps schools manage students, teachers, classes, attendance, grades, and school fees from a single platform.

The project was developed as part of the Principle of Structured Programming course and demonstrates the use of functions, loops, conditions, file handling, lists, dictionaries, and graphical user interfaces.

---

## Features

### User Management

* Login System
* Registration System
* Role-Based Access
* Admin Account Management

### Student Management

* Add Students
* View Students
* Search Students
* Generate Student IDs

### Teacher Management

* Add Teachers
* View Teachers

### Class Management

* Create Classes
* Assign Teachers
* Manage Rooms

### Attendance Management

* Mark Attendance
* Search Students
* Attendance Records

### Grade Management

* Record Grades
* Automatic Grade Calculation
* Student Performance Tracking

### Fee Management

* Record Payments
* Track Payment Status
* Fee Reports

### Database Management

* JSON File Storage
* Automatic Data Saving
* Automatic Data Loading
